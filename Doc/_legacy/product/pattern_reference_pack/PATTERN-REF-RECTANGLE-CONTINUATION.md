 # PATTERN-REF-RECTANGLE-CONTINUATION

## Status
Reference document for the intended V1 continuation-pattern scope.
This file is not runtime proof.

## Pattern identity
- Canonical id: `RECTANGLE_CONTINUATION`
- Domain label: `RectangleContinuation`
- Family: continuation pattern
- Bias: directional only after confirmed breakout
- Typical context: existing trend pauses into a bounded horizontal or near-horizontal trading rZange before resolving
Z
## Core reading
A rectangle continuation is a congestion phase where price oscillates between relatively flat support and resistance, creating a bounded range.
In standard technical-analysis education, rectangles are usually read as consolidation zones first and directional signals only after breakout confirmation.

## Structural conditions
A pattern candidate should not be treated as a rectangle continuation unless all of the following are materially present:
- a meaningful directional move exists before the range
- price then rotates inside a bounded support / resistance zone
- both boundaries are tested more than once
- the range visually dominates the local price action more than isolated wick noise
- the candidate still behaves like congestion, not like a trend channel with clearly sloping parallel bounds

## Situations to distinguish
### Uptrend before the rectangle
Interpretation is bullish continuation only after a confirmed upside breakout.
Before breakout, the rectangle remains neutral consolidation.

### Downtrend before the rectangle
Interpretation is bearish continuation only after a confirmed downside breakout.
Before breakout, the rectangle remains neutral consolidation.

### No meaningful prior trend
Do not force a continuation reading.
The structure may still be a rectangle, but not a continuation rectangle in the strict V1 sense.

## Detection reading
Useful observable facts:
- local highs cluster in a resistance zone
- local lows cluster in a support zone
- repeated back-and-forth movement inside the band is expected noise
- false break attempts are common

## Validation reading
A rectangle continuation becomes materially stronger when:
- breakout closes outside the range
- breakout direction matches the pre-pattern trend
- the market fails to re-enter the range after breakout
- supporting volume or other confirmation tools improve conviction

## Invalidation reading
A continuation reading weakens or fails when:
- breakout direction is opposite the intended continuation case
- price breaks out but quickly re-enters the rectangle
- post-breakout rejection is strong enough to negate the escape from the range
- the structure extends so long that the prior-trend continuation thesis becomes weak or stale

## Target logic
The most common educational measuring rule is to project the rectangle height from the breakout level.
That is a reference heuristic only.
It should be treated as a target framework, not as guaranteed market behavior.

## Risk and advice guidance
### When the pattern is only forming
Advice should remain observational.
No continuation claim should be stated as confirmed.

### When an upside continuation is being evaluated
Helpful guidance usually focuses on:
- waiting for a clean close beyond resistance
- checking whether the move holds outside the range
- monitoring whether the old resistance becomes support

### When a downside continuation is being evaluated
Helpful guidance usually focuses on:
- waiting for a clean close below support
- checking whether the move holds below the range
- monitoring whether the old support becomes resistance

### When breakout quality is weak
Advice should explicitly mention the false-breakout risk.

## Common mistakes to avoid
- treating every sideways range as a continuation rectangle
- assigning direction before confirmation
- ignoring the prior trend context
- ignoring re-entry into the range after breakout
- confusing a slightly sloped channel with a true rectangle congestion zone

## V1 wording guardrails
- Do not present the pattern as confirmed while price is still inside the range.
- Do not erase the possibility of false breakouts.
- Do not state target projection as certainty.
- Do not infer continuation when no prior trend is materially present.
