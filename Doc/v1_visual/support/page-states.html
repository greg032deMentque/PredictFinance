Les 4 familles d'états

Distinction stricte : une erreur technique n'est pas un état métier non-exécutable. Utilisez la barre du bas pour basculer.

Prêt

Contenu nominal affiché. C'est l'état par défaut.

Vide — aucune donnée pour cet utilisateur

Différent d'une erreur : il n'y a simplement rien à montrer encore.

Erreur récupérable (technique)

Échec temporaire. Action : réessayer.

État métier de premier rang
Non-exécutable métier

La requête est comprise mais non exécutable (ex. instrument hors périmètre, aucun pattern crédible). Ce n'est pas une erreur — c'est une issue métier expliquée.
