Règles de sécurité et légales
  • Message générique si email déjà utilisé : « Si cet email n'est pas encore enregistré, vous recevrez un lien de confirmation. » — jamais de confirmation d'existence.
  • Compte créé en statut PENDING — connexion refusée tant que non confirmé.
  • Les liens CGU et politique de confidentialité doivent exister avant ouverture en production (voir 08 §2 — bloquants légaux).
  • Aucune donnée bancaire, aucun numéro de téléphone, aucune vérification d'identité.
États à prévoir
  • Chargement : soumission en cours → bouton désactivé + spinner.
  • Erreur récupérable : service temporairement indisponible → message + réessayer.
  • Erreur validation : mot de passe trop faible / champs discordants → feedback inline par champ.
  • Non-exécutable : email déjà utilisé → message générique sans révéler l'existence.
  • CGU non cochées : bouton désactivé, invitation visible à cocher.
